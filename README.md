# 🚗 Leading Car

Welcome to **Leading Car**, an interactive and visually stunning image slider showcasing some of the world's most iconic supercars! 🏎️✨

## 🌟 Features

- **Smooth Image Slider**: Navigate through high-quality images of luxury cars with seamless transitions.
- **Interactive Controls**: Use arrow buttons, dot indicators, or keyboard arrows to browse slides.
- **Touch & Swipe Support**: Perfect for mobile devices with intuitive swipe gestures.
- **Auto-Play Mode**: Enjoy automatic slide progression every 5 seconds, with pause on hover.
- **Accessibility**: Fully accessible with ARIA labels and keyboard navigation.
- **Responsive Design**: Optimized for all screen sizes and devices.
